# ModePill

## Purpose

Displays a coloured pill for a channel mode (FM, DMR, YSF, etc.).

## Props

| Prop   | Type          | Notes            |
| ------ | ------------- | ---------------- |
| `mode` | `ChannelMode` | Internal mode id |
| `size` | Badge size    | Default `sm`     |
